# head

`<head/>` 엘리먼트에서 사용되는 요소. 구조 그대로 사용해도 좋을거 같다.

```javascript
const head = [
  {
    tag: 'title',
    value: 'title value',
  },
  {
    tag: 'meta',
    name: 'author',
    content: 'redgoose',
  },
  {
    tag: 'meta',
    name: 'description',
    content: 'description value',
  },
  {
    tag: 'link',
    rel: 'stylesheet',
    href: './style.css',
  },
]
```
